# 🚆 TrackSafe – Railway Alert (Lite Version)

> **Open Source PRD · MVP Edition**

## 🎯 Goal
Aplikasi mobile sederhana berbasis Flutter untuk:
- Share posisi kereta/user secara **realtime**
- Kirim **alert otomatis** saat terdeteksi potensi tabrakan atau berhenti mendadak

## ⚠️ Disclaimer
Bukan sistem resmi, tidak untuk operasi kereta sungguhan. Ini adalah sistem untuk edukasi, proof of concept, dan awareness.

## 🚀 Setup
1. Setup Firebase project & download `google-services.json` ke `mobile/android/app/`.
2. Jalankan aplikasi:
   ```bash
   cd mobile
   flutter create . # To generate platform folders
   flutter pub get
   flutter run
   ```
